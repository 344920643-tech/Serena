# Serena
# 华严寺数字化文旅ABS项目 - AR模拟器与定价模型

本仓库为**华严寺数字化文旅ABS产品设计报告**的配套技术实现部分，包含：
- 智能“复原”设备AR交互模拟器（报告3.1节核心展示）
- ABS产品现金流定价模型与蒙特卡洛风险模拟（报告5.2、5.4、5.5、5.6节核心计算）

---

## 📁 文件说明

| 文件名                        | 类型          | 作用说明 |
|-------------------------------|---------------|----------|
| **huayan_ar_simulator.py**    | Python 主程序 | **核心AR模拟器**。实现智能“复原”设备的实时AR效果，支持历史时期切换（辽→金→明→清→现代）和微观细节叠加（斗拱、梁柱、壁画、屋顶木构）。使用OpenCV实现透明叠加、时间轴、文字标注。 |
| **background.jpg**            | 背景图片      | AR模拟器的默认背景图（华严寺大殿内部或外观实景） |
| **overlay_liao.png**          | AR叠加层      | 辽代完整彩绘复原叠加图（主时期） |
| **overlay_jin.png**           | AR叠加层      | 金代复原叠加图 |
| **overlay_ming.png**          | AR叠加层      | 明代复原叠加图 |
| **overlay_beam.png**          | AR叠加层      | 梁柱榫卯结构细节AR叠加层 |
| **overlay_dougong.png**       | AR叠加层      | 斗拱细节AR叠加层 |
| **detail_mural.png**          | AR叠加层      | 壁画复原细节AR叠加层 |
| **detail_roof.png**           | AR叠加层      | 屋顶木构3D剖切AR叠加层 |
| **ABS产品定价模型.py**        | Python 模型   | ABS产品定价计算模型。包含加权平均票价、指数增长现金流预测、DCF定价、优先级规模计算等（对应报告5.2-5.4节）。 |
| **蒙特卡洛联合模拟.py**       | Python 模型   | 蒙特卡洛风险模拟脚本。对游客增长率与数字化转化率进行10,000次随机模拟，输出覆盖倍数分布、最差情景等（对应报告5.5-5.6节敏感性分析）。 |
| **README.md**                 | 说明文档      | 本文件 |
